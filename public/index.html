🚀 Solana NFT Minting API

Welcome to the Solana Native NFT Minting API. This service provides endpoints for minting, transferring, and managing NFTs on the Solana blockchain.

📋 Available Endpoints

POST /api/mint - Mint a new NFT
POST /api/verify-payment - Verify payment for minting
POST /api/generate-payment-qr - Generate payment QR code
GET /api/backend-wallet - Get backend wallet address
POST /api/transfer - Transfer NFT to another address
POST /api/update-royalty - Update NFT royalty settings
POST /api/transactions/create - Create new transaction
GET /api/transactions/:id/status - Check transaction status
